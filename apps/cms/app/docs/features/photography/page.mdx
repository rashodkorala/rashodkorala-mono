# Photography Management

Complete guide to managing your photography collection.

## Overview

The Photography feature allows you to manage your photo collection with:

- Single photo upload with file selector
- Bulk upload with JSON metadata
- AI-powered metadata generation
- Category organization
- Camera settings tracking
- Location and date information
- Tags for organization
- Featured photo flag

## Single Photo Upload

### Basic Upload

1. Navigate to **Photos** in the sidebar
2. Click **New Photo** button
3. Select an image file
4. Fill in photo details:
   - **Title**: Photo title
   - **Description**: Detailed description
   - **Category**: Photo category
   - **Location**: Where the photo was taken
   - **Date Taken**: Date the photo was captured
   - **Alt Text**: Accessibility description
5. Add camera settings (optional):
   - Aperture (f-stop)
   - Shutter speed
   - ISO
   - Focal length
   - Camera model
6. Add tags
7. Mark as **Featured** if desired
8. Click **Create Photo**

### AI-Powered Metadata

Use AI to automatically generate metadata:

1. Upload a photo
2. Click **AI Analyze** button
3. Wait for analysis (uses OpenAI GPT-4 Vision)
4. Review and edit the generated:
   - Title
   - Description
   - Category
   - Tags
   - Alt text
5. Make any adjustments
6. Click **Create Photo**

## Bulk Upload

Upload multiple photos at once with metadata from a JSON file:

1. Navigate to **Photos** in the sidebar
2. Click **Bulk Upload** button
3. **Step 1**: Upload metadata JSON file
4. **Step 2**: Select matching photo files
5. **Step 3**: Review and edit metadata
6. Click **Upload** to process all photos

### Metadata JSON Format

<CodeBlock language="json">
  {`{
  "photos": [
    {
      "title": "Sunset over Mountains",
      "description": "A beautiful sunset captured in the mountains",
      "category": "nature",
      "alt_text": "Sunset over mountain range with orange and pink sky",
      "image_path": "sunset-mountains.jpg",
      "location": "Yosemite, CA",
      "date_taken": "2024-01-15"
    }
  ]
}`}
</CodeBlock>

### Supported Categories

- `architecture` - Architecture & Heritage
- `nature` - Nature & Landscapes
- `street` - Street & Urban Life
- `travel` - Travel Photography
- `wildlife` - Wildlife
- `night` - Night & Lights
- `abstract` - Abstract / Artistic
- `interior_spaces` - Interior Spaces

## Editing Photos

1. Click the **Edit** button on any photo
2. Modify any fields
3. Replace the image if needed
4. Click **Update Photo**

## Photo Categories

Organize photos by category for easy browsing and filtering. Categories help structure your photography portfolio.

## Camera Settings

Track technical details of your photos:

- **Aperture**: f-stop value (e.g., f/2.8)
- **Shutter Speed**: Exposure time (e.g., 1/250s)
- **ISO**: Light sensitivity (e.g., ISO 400)
- **Focal Length**: Lens focal length (e.g., 50mm)
- **Camera Model**: Camera make and model

## Tags

Add multiple tags to photos for flexible organization. Tags allow for cross-category searching and filtering.

## Featured Photos

Mark photos as featured to highlight them on your frontend website. Featured photos typically appear in galleries or homepage sections.
